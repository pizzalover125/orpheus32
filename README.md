# orpheus32

<img width="276" height="501" alt="image" src="https://github.com/user-attachments/assets/c1c2e1a5-d478-4c72-b10c-ac01b4ad501a" />

An ESP32 devboard made for Hack Club. A rival to the famous Orpheus Pico. 

### features
- 23 GPIO
- Integrated WiFi + Bluetooth
- 2x UART
- 2x DAC (8-bit)
- SPI, I2C, I2S support
- User-addressable NeoPixel
